Setup

ready in 30 seconds

Three steps. No really, that's it.

1

Paste your OpenRouter key

Grab an API key from openrouter.ai and drop it in. That's your only credential — we keep nothing else.

2

Pick your model + directory

Enter the model name you want to use (Gemini 3.1 Pro, Claude 3.6 Sonnet, GLM 5.1 — whatever OpenRouter supports) and point juice at the folder you want to work in.

3

Start building 🚀

That's it. Tell juice what to build, reference files with @, pull in websites with #, and watch it go.
